# MCPaimon — FoliaMC

The FoliaMC platform implementation of the MCPaimon Minecraft AI framework. Hosts
the AI provider and manages MCExtension plugins on FoliaMC servers, respecting
Folia's regionised threading model.

Builds the runnable `MCAgentsFoliamc` plugin jar (Shadow) and publishes
`io.github.mcpaimon:foliamc` to GitHub Packages.

- Depends on `io.github.mcpaimon:api` and `io.github.mcpaimon:common` with static
  versions.

See [`AGENTS.md`](AGENTS.md) and the [`.agents/`](.agents) folder for contributor
instructions.
